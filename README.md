# Classimax – Classified Marketplace Website

## 📌 Overview

Classimax is a web-based classified marketplace platform that allows users to buy and sell products online. The system provides a simple, clean, and user-friendly interface for creating, browsing, and managing product listings efficiently.

---

## 🚀 Features

* Create and post product listings
* Browse and search available products
* View detailed product information
* Easy-to-use and clean user interface
* Responsive design for better user experience

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Python (Backend)
* MySQL (Database)

---

## 📂 Project Structure

```
classimax/
│── static/        # CSS, JS, images
│── templates/     # HTML files
│── app.py         # Main backend file
│── database/      # Database files
│── README.md
```

---

## ▶️ How to Run the Project

1. Clone the repository
2. Open the project in VS Code
3. Run the backend server (Python)
4. Open the browser and run the project locally
